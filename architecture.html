This the current architecture of my site

There are a lots of icons on this page but it is not difficult to explain what I try to do:

  1. All the files in this website are hosted in S3. So the user when goes to britcat. com is actually pulling those files from there.

  2. The feedback page contains a form with a submit button. The submit button connects via the Amazon API gateway with a Lambda function that takes the data from the form.

  3. The feedback page contains a form with a submit button. The submit button connects via the Amazon API gateway with two lambda functions that takes the data from the form.

  4. One Lambda function connects to the AWS SNS service that in turn sends an email to the business owner -me in this case-.


I am having trouble connecting the reCaptcha Form with the Lambda function so that's why I have decided to hide the feedback page for now.

Future plans

Connect GitHub with AWS so that I have continuous deployment and integration. In addition,I will include Amazon Cognito for user management and authentication.
